How to play

Cacho (Dudo/Perudo) live rooms. Create or join a room, claim a seat name, the host sets rules and starts rounds.

  1. Host clicks Firebase config… and pastes project config; creates Firestore DB.
  2. Click Create room, share the code/link.
  3. Each player clicks Join room, enters the code, and Claim/Update seat with their name.

For security, set Firestore Rules shown in the README section of this file (open source and search for “RULES”) so players can only read their own dice and only act on their turn.
